Download test

Both buttons call chrome.downloads.download to download an image with the SaveAs option set as true, however one doesn’t specify a filename and the other does.

(A) - will default to wherever the user most recently chose to “Save As” a file. Try saving this to the Desktop, then try it again, and you’ll see it default ot the Desktop.

(B) - will always default to the downloads directory.

There is no way to specify a filename while allowing “Save As” to default to most recently used directory.
